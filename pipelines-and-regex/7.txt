Во оваа задача ќе извлечеме IP адреси и домени од текстуален фајл logs.txt користејќи сложен регуларен израз. 
Регуларниот израз за IP адресите ќе опфаќа четири броја од 0 до 255, одделени со точки, 
додека за домените ќе биде покриена структурата на поддомени и основни домени. 
Командата ќе го користи grep со проширени регуларни изрази за да ги извлече IP адресите и домените од фајлот, 
а потоа ќе ги филтрираме со awk, така што ќе ги печатиме како IP адреси или домени во две различни категории, зависно од тоа што е извлечено.

Решение:
cat logs.txt | grep -Eo '([0-9]{1,3}\.){3}[0-9]{1,3}|([a-zA-Z0-9-]+\.)+[a-zA-Z]{2,}' | 
awk '{ if ($0 ~ /^[0-9]+\.[0-9]+\.[0-9]+\.[0-9]+$/) print "IP Address: " $0; else print "Domain: " $0 }'
